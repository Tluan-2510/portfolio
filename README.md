# 🚀 Nguyễn Thành Luân - Portfolio

A professional, high-performance portfolio website built with **Vite, Vanilla JS, and CSS**. This project features a modern "Antigravity" design concept with glassmorphism, dynamic animations, and a fully responsive layout.

## ✨ Key Features

- **🌓 Dynamic Theme**: High-fidelity Light and Dark mode with smart icon visibility.
- **📱 Fully Responsive**: Optimized for Mobile, Tablet, and Desktop viewports.
- **🛸 Antigravity UI**: Premium glassmorphism effects and smooth micro-animations.
- **⚡ High Performance**: Built with Vite for lightning-fast load times.
- **🎨 Custom Branding**: Personalized avatar and professional typography using Be Vietnam Pro.

## 🛠️ Tech Stack

- **Core**: HTML5, Vanilla JavaScript
- **Styling**: CSS3 (Custom Design System / CSS Variables)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Animations**: CSS Keyframes & Easing functions

## 📂 Project Structure

```text
portfolio/
├── src/
│   ├── components/      # UI Components (Header, Hero, BentoGrid, etc.)
│   ├── styles/          # Design tokens and global CSS
│   ├── utils/           # Helper functions and animation logic
│   └── main.js          # Application entry point
├── public/              # Static assets (Favicon, Icons, Avatar)
├── index.html           # Main HTML template
└── package.json         # Dependencies and scripts
```

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Tluan-2510/portfolio.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
